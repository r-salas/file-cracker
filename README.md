# file-cracker
Crack PDFs and ZIPs by brute force.

## Requirements
- Python 3.10+

## Installation
```console
$ pip install -r requirements.txt
```

## Usage
```console
$ python file_cracker.py --help
```

## Examples
```console
$ python file_cracker.py examples/sample.pdf
```

```
$ python file_cracker.py examples/sample.zip
```
